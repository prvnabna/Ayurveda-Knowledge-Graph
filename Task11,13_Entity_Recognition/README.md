# Task 11,13 - Entity Recognition and Semantic Tagging

## Objective

Develop a rule-based Named Entity Recognition (NER) pipeline for Ayurveda Knowledge Graph construction.

## Features

- Entity extraction using spaCy EntityRuler
- Alias matching
- Semantic tag enrichment
- Automated annotation generation
- Precision, Recall and F1 evaluation

## Datasets

- Ashwagandha
- Clinical Evaluation
- Ayurveda Management of Diabetes

## Technologies

- Python
- spaCy
- Pandas
- Google Colab
- Excel

## Results

| Dataset | Precision | Recall | F1 |
|----------|----------|----------|----------|
| Ashwagandha | 83.94 | 83.08 | 83.51 |
| Clinical | 54.05 | 57.14 | 55.56 |
| Diabetes | 81.31 | 92.55 | 86.57 |
